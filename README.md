# Home Cybersecurity Lab

## Overview
This project demonstrates a home cybersecurity lab built to simulate attacker and victim activity in a safe, isolated environment. The lab focuses on network reconnaissance, traffic analysis, and SIEM-based log monitoring.

## Environment
- VirtualBox
- Windows 10 (Victim)
- Kali Linux (Attacker)
- Host-only Networking
- Wireshark
- Nmap
- Splunk Enterprise SIEM

## Objectives
- Build an isolated virtual lab
- Simulate attacker reconnaissance
- Capture and analyze network traffic
- Monitor Windows logs using a SIEM

## Skills Demonstrated
- Networking fundamentals
- Windows and Linux administration
- Network traffic analysis
- Log ingestion and SIEM monitoring
